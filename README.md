# snakemake_noncoding
This pipeline incorporates ML/DL based tools to predict the pathogenicity of non-coding variants from the VCF file.
The pipeline includes following tools and databases-
Tools-
Cscape
Candra
Funseq2
Annovar
Cadd
Fathom
Sift

Databases-
Refgene
Gewascatalog
Avsnp
Clinvar
gnomad
dbnsfp
